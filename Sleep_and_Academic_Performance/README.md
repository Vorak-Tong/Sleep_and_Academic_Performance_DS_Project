# Sleep and Academic Performance

## Project Overview
This project aims to analyze the relationship between students' sleep patterns and their academic performance. By using data analysis and machine learning techniques, we explore how sleep duration and other lifestyle factors influence GPA.

---

## Objectives
- Analyze the relationship between sleep habits and academic performance  
- Identify key factors affecting GPA  
- Build a machine learning model to predict academic performance  

---

## Dataset
The dataset contains student lifestyle and academic information, including:
- Sleep hours per day  
- Study hours per day  
- Stress level  
- GPA (target variable)  

The data is obtained from publicly available sources and cleaned before analysis.

---

## Project Workflow

### 1. Data Cleaning
- Removed unnecessary columns (e.g., student_id)  
- Handled missing values and duplicates  
- Verified data types  

### 2. Exploratory Data Analysis (EDA)
- Distribution analysis (sleep hours, GPA, study hours)  
- Relationship analysis (sleep vs GPA, study vs GPA)  
- Stress level analysis  
- Correlation analysis (heatmap)  

### 3. Modeling
- Split dataset into training and testing sets  
- Applied machine learning models (e.g., Linear Regression / Logistic Regression / Decision Tree)  

### 4. Evaluation
- Evaluated model performance using appropriate metrics  

---

## Key Findings
- Sleep duration shows a positive relationship with GPA  
- Study hours have the strongest influence on academic performance  
- High stress levels are associated with lower GPA
